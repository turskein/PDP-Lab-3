# PDP-Lab-3
Repositorio del laboratorio numero 3 de paradigmas de programación

Para poder iniciar el proyecto a través de gradle tan solo es necesario abrir el
archivo llamado "abrir.bat". En el caso de querer ejecutar funcionalidades que
vienen incorporadas en el proyecto se deben ingresar las distintas entradas que
se solicitan.

Dentro del proyecto es han generado 2 plataformas. En la plataforma ParadigmaDocs
se presenta un desarrollo de documentos, versions, accesos y comentarios
realizados por lo usuarios presentes en él, se debe mencionar que estos testeos
realizados están presentes en la carpeta de códigoFuente>UI en la función llamada 
testGenerator.
